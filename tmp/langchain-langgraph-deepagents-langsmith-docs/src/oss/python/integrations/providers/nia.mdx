[Nia](https://trynia.ai) is a search and index API that continuously provides context from docs, research papers, datasets, codebases, and more—so agents never rely on stale data. Scalable, 5x cheaper, and reliable.

## Installation and setup

<CodeGroup>
```bash pip
pip install langchain-nia
```

```bash uv
uv add langchain-nia
```
</CodeGroup>

Set your API key:

```bash
export NIA_API_KEY="nk_..."
```

## Tools

The `langchain-nia` package provides 20 tools organized into a configurable toolkit.

See the [Nia Toolkit integration guide](/oss/integrations/tools/nia) for full details, usage examples, and the complete list of available tools.

```python
from langchain_nia import NiaToolkit

toolkit = NiaToolkit()
tools = toolkit.get_tools()  # 20 tools
```
