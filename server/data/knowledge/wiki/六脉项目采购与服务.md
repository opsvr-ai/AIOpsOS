# 六脉项目采购与服务

## 概述

[[六脉智能运维数智平台研发项目]]为新增内部创新项目，涉及一定程度的采购服务。项目计划采购外部咨询服务和研发力量，以弥补内部技术储备与人力不足，同时保障自主可控。

## 采购计划

- **服务采购**：是（采购）
- **预计采购金额**：100（万元）
- **采购内容**：外部咨询服务、研发力量支持

## 合作模式

项目采用“以联盟人员为主，采购外部咨询服务和研发力量为辅”的合作模式。该模式旨在避免产品捆绑和技术无法延续的问题，同时确保联盟对核心技术的自主可控及项目的可持续性。该模式也写入了[[六脉项目工作风险分析]]中的风险应对措施（技术储备及人力投入不足风险）。
