# LangChain

LangChain是一个开源框架（Python/JS双版本），专门用于构建大语言模型（LLM）应用程序，核心定位是“连接大模型与外部资源/工具的桥梁”。^[raw/公司级创新申报材料-生产运维部_v0407-5.md]

## 关键特点

- **模块化设计**：将LLM能力与外部资源、业务逻辑进行有机整合。
- **统一接口**：为不同LLM提供商（如OpenAI、通义千问、月之暗面等）提供标准化接口。
- **解决核心问题**：突破LLM的固定上下文窗口限制，弥补大模型的局限性（知识更新滞后、API调用能力等）。

## 核心模块

- **Models**：统一的大模型接口。
- **Prompts**：提示词模板管理。
- **Chains**：工作流程设计。
- **Memory**：上下文记忆存储。
- **Tools**：外部工具调用。
- **Agents**：AI自动决策系统。
- **Retrieval**：知识检索（RAG技术）。

## 在项目中的应用

在 [[六脉智能运维数智平台研发项目]] 中，LangChain 作为底层框架，用于将大模型能力与运维工具、数据源对接，支撑智能体客户端的告警分析与处置建议生成。^[raw/公司级创新申报材料-生产运维部_v0407-5.md]
