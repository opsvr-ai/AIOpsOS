# 六脉项目主要工作内容

六脉项目（[[六脉智能运维数智平台研发项目]]）核心是构建融合AI能力的智能运维数智平台，打造运维决策与数据中枢，重构传统运维工作模式，破解人力不足、效率偏低、被动响应等痛点，支撑21+6关键业务系统值守与运维服务赋能。主要包括以下三个部分：^[raw/公司级创新申报材料-生产运维部_v0407-5.md]

## 智能运维数智平台

围绕“**[[运维中枢神经大脑]] + + [[智能体专家矩阵]]**”联动模式，构建多智能体协同平台。服务端集成多智能体协同、Skills动态发现等核心能力，实现生产系统运行自主扫描、隐患预测及辅助处置前置，为**[[故障取证与根因定界]]**、应急协同以及日常运维等场景提供决策支撑，破解运维数据利用低效问题。^[raw/公司级创新申报材料-生产运维部_v0407-5.md]

## 智能体客户端

面向多场景构建融合式AI智能体客户端（参见[[agent-client]]），根据业务场景差异化集成对应算法与运维能力。配合平台实现全链路数据自主采集与分析，实时监控各层级运维对象，自动分析异常、推送告警并生成处置建议，打通运维数据采集“最后一公里”，支撑运行态信息收集分析、故障预警等核心业务能力落地。^[raw/公司级创新申报材料-生产运维部_v0407-5.md]

## 高质量运维场景建设

聚焦业务目标的故障取证与根因定界、应急协同核心需求，打造核心基础能力并向下赋能日常运维，探索破解故障处置低效痛点路径，支撑21+6关键业务系统故障定界目标实现。本方案通过系统重构与流程优化，颠覆传统运维模式，实现运维流程再造，助力提升运维质效、降低人力成本，沉淀AIOps技术资产。^[raw/公司级创新申报材料-生产运维部_v0407-5.md]
