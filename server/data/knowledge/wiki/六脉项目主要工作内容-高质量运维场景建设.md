# 六脉项目主要工作内容-高质量运维场景建设

作为[[六脉智能运维数智平台研发项目]]的重要组成部分，**高质量运维场景建设**聚焦业务目标的故障取证与根因定界、应急协同核心需求，打造核心基础能力并向下赋能日常运维。

## 关键场景

- **故障取证与根因定界**：通过[[故障取证与根因定界]]流程，结合[[六脉项目故障定界大小模型协同机制]]，实现快速定位。
- **应急协同**：利用[[fault-emergency-agent-collaboration]]中的智能体协同能力，优化多岗位信息共享。
- **日常运维赋能**：将故障场景中验证的能力下沉至变更管理、告警压缩、巡检自动化等日常操作。

## 目标效果

- 支撑[[21+6关键业务系统]]故障定界目标实现。
- 探索破解故障处置低效痛点路径。
- 为后续全面推广积累经验，最终实现[[六脉项目赋能联盟成员行生态]]。

该场景建设依赖于[[六项目主要工作内容-智能运维数智平台详细]]和[[六项目主要工作内容-智能体客户端详细]]提供的底层能力。
