# 智能运维数智平台

**智能运维数智平台**是[[六脉智能运维数智平台研发项目]]的核心产出，采用“运维中枢神经大脑 + 智能体客户端 + 智能体专家矩阵”的混合架构，旨在实现运维流程的智能化再造。

## 核心架构
### 运维中枢神经大脑
- 多智能体协同调度中心
- 集成[[Harness工程]]理念，提供安全约束、校验、执行、记忆、错误自愈等能力
- 基于[[LangChain]]和[[DeepAgents]]框架构建

### 智能体客户端
- 面向多场景的融合式AI客户端
- 根据业务场景差异化集成算法与运维能力
- 实现全链路数据自主采集与分析，实时监控运维对象

### 智能体专家矩阵
- 为联盟重要业务系统定制专属运维数字专家智能体
- 各智能体明确职责边界，在统一调度下协同工作
- 探索“大脑+小脑”运维场景新模式

## 关键技术
- **大模型 + 小模型协同**：小模型负责算法提纯筛选关键数据，大模型负责逻辑推理和根因研判
- **多智能体协同**：通过DeepAgents框架实现自主规划、子代理生成、对话压缩等
- **安全栅栏**：对执行类增加人工确认，智能体只负责分析和决策，不直接执行操作

## 应用场景
- 故障取证与根因定界
- 应急协同
- 日常运维赋能
- 21+6关键业务系统值守
