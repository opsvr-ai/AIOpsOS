# 六脉项目主要工作内容-智能运维数智平台详细

作为[[六脉智能运维数智平台研发项目]]的核心工作内容，本项目构建**智能运维数智平台**，采用“[[运维中枢神经大脑]] + + [[智能体专家矩阵]]”联动模式，打造多智能体协同平台。

## 平台核心能力

- **服务端能力**：集成多智能体协同、[[Skills动态发现]]等核心能力，实现生产系统运行自主扫描、隐患预测及辅助处置前置。
- **数据支撑**：为故障取证与根因定界、应急协同以及日常运维等场景提供决策支撑，破解运维数据利用低效问题。
- **模式创新**：在服务端融入[[Harness工程]]理念，通过[[DeepAgents]]框架实现智能体的稳定运行与自我迭代。

## 核心架构

平台采用分层架构：
- **下层**：LangChain框架与LangGraph运行时，提供AI智能体基础能力。
- **上层**：DeepAgents应用层工具集，实现开箱即用的智能体编排。
- **业务层**：对接[[21+6关键业务系统]]，形成运维决策中枢。

此平台的构建是和的基础。
