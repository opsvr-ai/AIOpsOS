# 六脉项目主要工作内容-智能体客户端详细

在[[六脉智能运维数智平台研发项目]]中，**智能体客户端**（[[agent-client]]）扮演着数据采集与初步分析的关键角色。本项目将面向多场景构建融合式AI智能体客户端。

## 核心功能

- **差异化集成**：根据业务场景差异化集成对应算法与运维能力，如日志分析、指标监控、调用链追踪等。
- **全链路自主采集**：配合[[智能运维数智平台]]实现全链路数据自主采集与分析，实时监控各层级运维对象。
- **智能分析**：自动分析异常、推送告警并生成处置建议，打通运维数据采集“最后一公里”。
- **业务支撑**：支撑运行态信息收集分析、故障预警等核心业务能力落地。

## 设计特点

智能体客户端采用轻量化设计，可部署于各业务系统侧，与平台服务端保持实时通信。其内部集成了小模型算法提纯能力（见[[大小模型协同]]），实现边缘端快速数据预处理。

此客户端与[[六项目主要工作内容-智能运维数智平台详细]]中的服务端配合，共同构建“大脑+小脑”协同模式（详见[[双向认知沟通模式]]）。
