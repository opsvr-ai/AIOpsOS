# DeepAgents

DeepAgents 是 [[LangChain]] 团队推出的 AI 代理框架，基于 LangChain 和 LangGraph 构建，专门开发能够执行复杂、开放式任务并在较长时间跨度内持续运行的“深层智能体”。^[raw/公司级创新申报材料-生产运维部_v0407-5.md]

## 核心特点

- **分层架构**：在 LangChain（框架层）和 LangGraph（运行时层）基础上构建应用层。
- **深度能力**：提供规划工具、文件系统后端、子代理生成能力。
- **长期运行**：支持长期规划、记忆管理与自主迭代。
- **解决“肤浅”问题**：传统 LLM 工具调用代理在处理长周期任务时表现“肤浅”，DeepAgents 专门解决此问题。

## 核心能力

- 自主规划：分解复杂任务，制定执行计划。
- 文件系统访问：读写和管理文件。
- 子代理生成：创建和管理子代理协同完成任务。
- 对话压缩：有效管理长对话上下文。
- 结果回收：确保任务执行结果的可靠收集。

## 在项目中的应用

在 [[六脉智能运维数智平台研发项目]] 中，DeepAgents 作为多智能体框架，支撑运维数字专家智能体的创建与协同工作，实现故障根因定界和应急协同场景的智能化。^[raw/公司级创新申报材料-生产运维部_v0407-5.md]

## 技术定位

- LangChain：提供 AI 智能体框架（核心抽象能力）。
- LangGraph：聚焦运行时层面（部署阶段的稳定性与可恢复性）。
- DeepAgents：提供开箱即用的应用层工具集（降低 AI 智能体开发门槛）。
