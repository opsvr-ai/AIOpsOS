# OpenClaw

OpenClaw 是《[[六脉智能运维数智平台研发项目]]》申报材料中提及的一项多智能体协同技术，被认为推动 AI 向高阶模式迭代的代表性技术之一。^[raw/公司级创新申报材料-生产运维部_v0407-5.md]

## 当前知识
- 在项目工作背景中，提到“以 OpenClaw 为代表的多智能体协同技术推动 AI 向高阶模式迭代”^[raw/公司级创新申报材料-生产运维部_v0407-5.md]
- 生产运维部曾针对 OpenClaw 技术在运维领域的应用开展初步探索，并与后续使用的 [[DeepAgents]] 落地路线进行对比。^[raw/公司级创新申报材料-生产运维部_v0407-5.md]
- 该名称可能是 OpenAI 旗下某项多智能体框架（如 Swarm、Operator 等）的笔译或内部代号，但原始材料明确指出“OpenClaw”。

## 与项目的关系
OpenClaw 是项目前期技术预研的参考方向之一，最终项目中采用的技术路线为 [[Harness工程]]、[[LangChain]] 生态下的 [[DeepAgents]] 框架。

## 开放问题
- OpenClaw 的具体技术形态、是否可公开获取尚不明确。
- 与 DeepAgents 相比，OpenClaw 在运维场景中的优劣有待验证。
