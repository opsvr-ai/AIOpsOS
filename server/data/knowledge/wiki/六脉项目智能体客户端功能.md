# 六脉项目智能体客户端功能

## 概述

智能体客户端是[[六脉智能运维数智平台]]架构中的重要组成部分，配合[[运维中枢神经大脑]]实现运维数据的全链路自主采集与分析。它面向多场景构建融合式AI客户端，根据业务场景差异化集成对应算法与运维能力。

## 核心功能

- **全链路数据采集**：实时监控各层级运维对象，自动采集运行态数据，打通运维数据采集的“最后一公里”。
- **异常分析与告警**：自动分析异常状态，推送告警信息并生成处置建议。
- **故障预警支持**：支撑运行态信息收集分析、故障预警等核心业务能力落地。
- **差异化集成**：根据不同业务系统场景，灵活集成对应算法与运维能力，实现定制化智能服务。

## 在平台中的角色

客户端作为“智能体客户端”，与[[智能体专家矩阵]]协同工作。它负责执行[[运维中枢神经大脑]]下发的任务，并自主完成数据层面的处理，将结果反馈给中枢。这种设计突破了传统“客户端-服务端”单向运行模式，形成了双向认知沟通模式。
