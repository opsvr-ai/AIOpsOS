# 六脉项目工作目标-聚焦故障应急场景

作为[[六脉智能运维数智平台研发项目]]的关键工作目标之一，本项目聚焦**故障应急场景**，通过AI多智能体技术提升处置效率，重点解决传统运维中故障根因定界慢、定位难以及应急协同不畅的核心痛点。

## 核心措施

- **大小模型协同**：深度融合[[大小模型协同]]技术。小模型负责算法提纯，快速筛选提炼故障关键数据；大模型侧重逻辑推理，辅助研判故障根因，快速给出处置建议。
- **应急流程融入**：尝试将[[运维数字专家智能体]]融入应急处置流程，重点解决应急过程中多岗位信息协同不畅的问题，提升应急协同效率。

## 预期收益

通过该目标的实现，可显著缩短故障平均修复时间（MTTR），降低人工依赖，并为[[六脉项目工作目标-立足路径实现验证]]提供实际案例数据支撑。

## 关联场景

该目标直接服务于[[21+6关键业务系统]]的故障定界目标，并与[[故障取证与根因定界]]、[[fault-emergency-agent-collaboration]]等页面描述的流程相衔接。
