# 六脉项目基本信息

本项目为《六脉》智能运维数智平台研发项目，由生产运维部申报，为**新增**类型创新项目。^[raw/公司级创新申报材料-生产运维部_v0407-5.md]

| 字段 | 内容 |
|------|------|
| **创新名称** | 【六脉】智能运维数智平台研发项目 |
| **申报部门** | 生产运维部 |
| **申报类型** | 新增 |
| **创新类型** | 内部创新 |
| **服务采购** | 采购，预计采购金额 100 万元 |
| **负责人** | （未填写） |

该项目属于内部创新，不涉及金融创新。计划通过采购外部服务（预计金额100万元）来辅助平台建设。^[raw/公司级创新申报材料-生产运维部_v0407-5.md]
