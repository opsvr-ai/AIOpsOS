线性三步向导

按顺序推进，每步独立完成。必配项完成后可直接进入平台，LDAP 和渠道可后续配置。

请确认以下几点

A

纯步骤式推进

必须按 1→2→3 顺序（但 Step1 完成后可直接"进入平台"，跳过 2、3）。当前推荐方案。

B

Tab 页签式切换

三个配置Tab可自由切换，大模型配置完成前其他Tab置灰。完成所有配置统一提交。

C

单页面长表单

所有配置在一个长页面中，Section 分区，顶部有进度指示器。简单直接但页面较长。

关于"跳过"逻辑

P1

必配项完成后可随时"进入平台"

Step 1（大模型）完成后就出现"进入平台"按钮。Step 2/3 提供"跳过，以后配置"按钮。后续可在系统管理页面完成。推荐。

P2

全部完成才能进入

三步必须全部勾选"已完成"或显式跳过才能进入平台。确保管理员不会遗漏关键配置。

视觉风格

V1

深色科技风

深蓝/深灰背景 + 发光边框 + 粒子/网格动效。类似运维监控大屏的科技感。

V2

浅色科技风（推荐）

与登录页面一致的浅色渐变 + 微质感卡片 + 精细阴影 + 淡入淡出动效。保持平台UI一致性，同时加入科技点缀（点阵背景、细线网格）。
