工具市场页面 — 布局方案

智能体页面已选：统计概览 + 卡片/表格切换。工具市场是否沿用相同模式，还是有不同需求？

A

与智能体页面一致

统计概览 + 卡片/表格切换。顶部显示工具总数/启用/内置统计，卡片网格默认展示，可切换到表格。保持两个页面设计语言统一。

B

市场风格

去掉统计概览，强化分类 Tab 导航（全部/内置/技能/插件/MCP）。卡片更大、展示更多描述信息，类似插件市场的浏览体验。支持按分类、标签筛选。

C

按类型分视图

内置工具用紧凑列表（不可编辑），Skill/Plugin/MCP 用市场卡片。每种类型有独立的展示逻辑，强调"内置工具是系统能力，市场工具是可扩展生态"的区别。
