内置标记行为规则

内置智能体

  • 显示"内置"徽章 (金色)
  • 不可删除（按钮隐藏）
  • 不可停用（始终活跃）
  • 可编辑 user_prompt 和工具关联
  • 仅 admin 可编辑 system_prompt

内置工具

  • 显示"内置"徽章 (金色)
  • 不可删除
  • 不可停用
  • 不可编辑（名称/描述/配置锁定）

实施范围

后端 (server/)

  • Agent / Tool 模型增加 is_builtin 字段 (Boolean, default False)
  • 种子数据标记 is_builtin=True
  • API 响应 schema 增加 is_builtin
  • 删除/停用/编辑端点增加内置保护校验

前端 (web/)

  • 智能体页面：统计概览栏 + 卡片/表格切换 + 内置徽章 + 操作按钮条件隐藏
  • 工具市场：统计概览栏 + 卡片/表格切换 + 内置徽章 + 操作按钮条件隐藏
