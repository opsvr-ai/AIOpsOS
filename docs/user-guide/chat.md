# AI 智能体对话

## 创建新会话

进入「运维中心 → 对话」，点击「新建会话」。系统会为每个会话分配唯一 ID，自动关联记忆和上下文。

## 与智能体对话

AIOpsOS 的智能体是 LangGraph 驱动的多步骤推理系统，主流程为：

1. **意图识别** — 分析用户输入，识别运维意图
2. **规划** — 制定执行计划，拆分子任务
3. **路由** — 根据任务类型分发到对应子智能体
4. **执行** — 调用工具执行具体操作
5. **综合** — 汇总子任务结果，形成最终回答
6. **反思** — 评估结果质量，必要时重试

## 子智能体

系统内置 4 个子智能体，会根据任务自动调用：

| 子智能体 | 职责 |
|----------|------|
| **知识库智能体** | 搜索知识库，查找运维文档和经验 |
| **监控智能体** | 查询监控指标，获取系统状态 |
| **运维智能体** | 执行运维操作（重启服务、查看日志等） |
| **分析智能体** | 深度分析问题根因 |

## 流式响应

对话采用 SSE（Server-Sent Events）流式传输，你可以实时看到意图识别结果、执行计划、工具调用过程和最终回答。

## 记忆系统

每次对话结束后，系统会自动：

1. **个人记忆** — 抽取操作细节、踩坑点、配置要点，关联到当前会话
2. **团队记忆** — 提炼通用经验、标准解决思路（已脱敏）

在后续对话中，相关记忆会自动注入上下文，帮助智能体更好理解你的需求。
