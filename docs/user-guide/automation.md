# 自动化

## 定时任务

定时任务基于 Celery Beat + RedBeat 实现，支持动态 Cron 管理。

### 创建定时任务

1. 进入「AI中心 → 定时任务」
2. 点击「新建」
3. 配置 Cron 表达式（支持可视化 Cron 编辑器）
4. 关联场景
5. 设置是否启用
6. 保存

### Cron 表达式示例

| 表达式 | 说明 |
|--------|------|
| `*/5 * * * *` | 每 5 分钟 |
| `0 * * * *` | 每小时整点 |
| `0 9 * * 1-5` | 工作日早 9 点 |
| `0 2 * * 0` | 每周日凌晨 2 点 |

## 触发规则

触发规则用于自动响应外部事件（如告警），匹配条件后自动触发指定场景。

### 规则配置

- **条件匹配** — 支持简单模式（字段匹配）和表达式模式（自定义逻辑）
- **关联场景** — 触发时执行的场景
- **频率控制** — 防止同一事件重复触发

## 执行历史

每次定时任务或触发规则执行后，记录包括：
- 执行时间
- 执行状态（成功/失败）
- 执行摘要
- 关联的会话 ID（可点击跳转查看详情）
