# 快速开始

## 登录

访问系统地址，默认管理员账号：
- 用户名：`admin`
- 密码：`admin123`

首次登录后请修改默认密码。

## 界面布局

登录后进入主界面，左侧为导航菜单，分为三大区域：

| 区域 | 说明 |
|------|------|
| **运维中心** | 总览、AI 对话、告警中心、场景运维、自动化、历史记录 |
| **AI中心** | 智能体管理、工具注册、知识库、定时任务、记忆管理 |
| **控制中心** | 消息渠道、身份与权限、系统管理 |

## 第一次对话

1. 点击左侧「运维中心 → 对话」
2. 点击「新建会话」按钮
3. 在输入框输入运维问题，例如："帮我查一下系统负载情况"
4. 智能体会分析意图、制定计划、执行工具调用，并返回结果
5. 对话结束后，系统会自动抽取关键信息保存为个人记忆

## 暗色主题

点击右上角月亮图标切换暗色/亮色主题。
